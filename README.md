# EventAI - AI-агент-куратор Demo Day

## Задача

Demo Day AI Talent Hub (ИТМО) - 330 проектов, 10 залов, 2 дня. Три проблемы:

- Гость видит ~16% программы и выбирает вслепую
- Организатор кластеризует вручную; 80 студентов выпали из расписания
- Эксперты приходят без информации, не могут выбрать слоты, оценки собираются в таблицах

EventAI - агентная система для обоих контуров: участникам - персональная программа и Q&A, организаторам - автоматизация подготовки.

## Целевая аудитория

- **Внешние гости** (партнеры, инвесторы, HR, менторы) - не могут найти релевантные проекты среди сотен
- **Организаторы** (~5 чел.) - импорт, кластеризация, расписание, рассылки - все вручную
- **Эксперты** (~50 чел.) - нет инструмента для выбора слотов и выставления оценок
- **Бизнес-партнеры** (~15-20 чел.) - нет поиска проектов под задачу, постсобытийное сопровождение хаотичное

## Что делает PoC

### Telegram-бот (агентская часть)

- **Профилирование:** LLM уточняет роль/интересы/цели за 2-3 реплики - структурированный профиль (JSONB)
- **Рекомендации:** эмбеддинг (Gemini) - Qdrant top-30 - переранжирование с учетом расписания - LLM-резюме - top-15
- **Анализ артефактов:** парсинг презентаций (PPTX/PDF) и GitHub-репозиториев при импорте; агент отвечает по содержимому проекта
- **Агент с вызовом инструментов:** карточка проекта, сравнение 2-5 проектов, подсказки для Q&A, пакет контактов, воронка для бизнеса, вопрос организатору
- **Эксперты в боте:** выбор слотов доступности, оценки проектов после защиты
- **Канал бот-админка:** вопросы участников - админка - ответ - бот
- **Плавная деградация:** LLM недоступна - ответы без LLM, Qdrant недоступен - пересечение тегов, ключ заблокирован - ротация + запасная модель

### Веб-админка (оргконтур)

- Импорт Excel + валидация + дедупликация (SHA-256)
- AI-кластеризация: LLM - валидатор - автоисправления - перетаскивание - аудит
- Распределение экспертов: IDF-взвешенный нечеткий матчинг + LLM-смежные теги
- Расписание: генерация слотов, перетаскивание, экспорт .ics/.xlsx
- Рассылки, напоминания, брифинги, дашборд покрытия
- Входящие вопросы от участников - ответы из админки

### Тестирование

- Модульные/интеграционные: 317 тестов (217 фронтенд + 100 бэкенд)
- Сквозные: онбординг - профилирование - рекомендации - вызов инструментов

## За рамками PoC

- Организация встреч 1:1 (только запрос контакта автора)
- Самостоятельная загрузка артефактов студентами
- Автоматическая фильтрация персональных данных
- Ограничение частоты запросов к боту
- Локальная LLM
